Guidelines to set up your own websites

You can set up your own website by just uploading your web pages to the share folder Web. The web server is optimized for hosting PHP-MySQL-based sites and you can simply upload your web pages to the network share Web by the following 2 methods:


  • Windows Mapped Shares

  • FTP Upload

Windows Mapped Shares

By mapping the network shares (Web) on the NAS server to a drive folder in Windows and you may refer to the step-by-step guide below.

Step 1 > Right click on My Computer and select 'Map Network Drive...'

Step 2 > Select a Drive letter that you wish to map Web to.

Step 3 > Now type in the username and Password to grant access on the Web share.

Step 4 > Drive Y: is now mapped to Web successfully and can be accessed just like a
drive Folder on your Windows PC.

Step 5 > You may now copy the website contents into this Folder and they will instantly
be available to access from the internet or within your home network.

FTP Upload

Connect to the Web share via any FTP client of your choice and type in the username
and password when prompted.

Step 1 > Connect to the Web share via any FTP client of your choice and type in
the username and password when prompted.

Step 2 > Once logged in simply drag and drop the website contents you wish to upload
to the Web FTP directory.

Step 3 > That's pretty much all you have to do and your web site can now be accessed
directly on the Internel.
