Bevor Sie starten Dieses Dokument ist sowohl eine kurze Einführung, als auch ein Ort, an dem Sie Dinge ausprobieren können. Es wird Ihnen gezeigt, wie Sie in 6 Schritten eine toll aussehende Website erstellen. 1) Öffnen Sie die Design-Galerie und wählen Sie ein Design aus. Gut, das muss nicht mehr gemacht werden, weil dieses Dokument bereits geöffnet ist. Es ist im Vorlagen-Design „Beta“. Sie können alle Vorlagen sehen, indem Sie im Menü folgendes auswählen:   Datei -> Neu aus der Design-Galerie Vorschau der Website: Sie können jederzeit Datei -> Website-Vorschau auswählen oder einfach auf dieses Symbol in der oberen Leiste drücken. Versuchen Sie es mal, um zu sehen, wie diese Seite in einem Browser aussieht. Testen Sie im Vorschau-Fenster die oberen Schaltflächen und bewegen Sie die Objekte, wie dieses Foto, um verschiedene Effekte zu sehen. 2) Grundfarben bearbeiten Bewegen Sie den Mauszeiger ans linke Ende der Farbleiste unten und klicken Sie auf eines der quadratischen Farbfelder. Dies sind die Grundfarben dieser Beispiel-Website. Klicken Sie auf eine Farbe und wählen Sie „Bearbeiten“. Dadurch öffnet sich der Farbeditor, in dem Sie verschiedene Farben auswählen können. Wir empfehlen, dass Grundfarbe 1 und 2 ähnliche oder zueinander passende Farben sind. Alle Elemente in diesem Design haben Grundfarben. 3) Text auf Ihrer Website bearbeiten Klicken Sie links auf das große T-Symbol (das Text-Werkzeug). Sie können nun auf jeden Text klicken und diesen bearbeiten. Ersetzen Sie z. B. die Überschrift mit Ihrer eigenen. Im Text-Werkzeug können Sie über die Leiste oben die Schriftart, -größe usw. einstellen. Das ABC-Symbol ist für die Rechtschreibkontrolle. Die Textbearbeitung funktioniert wie in einer Textverarbeitung. Ziehen Sie z. B. um Text auszuwählen. So ändern Sie auch die Aufschrift einer Schaltfläche. Einige Schaltflächen sind „dehnbar“ und für beliebig langen Text geeignet. Nehmen Sie z. B. die Schaltfläche links. Wenn Sie die Verlinkung der Schaltfläche ändern möchten, wählen Sie diese aus und klicken Sie auf das Ketten-Symbol oben. Um Text zu verlinken, wählen Sie diesen einfach im Text- Werkzeug aus und benutzen Sie dieselbe Schaltfläche. 4) Objekte neu anordnen Nun verschieben und skalieren wir Objekte. Wählen Sie das Auswahl-Werkzeug (der große Pfeil ganz oben) und ziehen Sie am Foto oben rechts auf dieser Seite. Sie sehen, dass der Text um es fließt. Verschieben Sie nun die Überschriften oder einige Schaltflächen und starten Sie die Vorschau. Was Sie in Xtreme Web Designer sehen, wird genau so auf Ihrer Website dargestellt. Egal was, egal wo. Sie skalieren ein Objekt, indem Sie an einem der Eckanfasser um das Objekt ziehen. Versuchen Sie es mit dieser Blume unter dem Text. Klicken Sie mit dem Auswahl-Pfeil auf sie und ziehen Sie an einem Eckanfasser. Um ein Objekt zu drehen, klicken Sie erneut darauf und es bekommt Drehanfasser. Ziehen Sie an diesen, um das Objekt zu drehen. Versuchen Sie dies an den Blüten oder sogar mit einer Text-Überschrift. So bewegen, skalieren oder drehen Sie so ziemlich alle Objekte auf Ihrer Seite.
