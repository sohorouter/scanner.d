 Startseite

Hier finden Sie eine Anleitung zum Aktivieren Ihrer Cookie-Einstellungen in Ihrem Browser:

- Internet Explorer 5
- Internet Explorer 6
- Mozilla Firefox
- Opera

Sollte sich in der englischen Version ein Begriff unterscheiden, so wird der englische Begriff dahinter in Klammern übersetzt.

schließen
Internet Explorer 5 schließen

1. Klicken Sie im Internet Explorer zunächst auf den Menüpunkt Extras (Tools). Es erscheint ein Pulldown-Menü, in dem Sie auf Internetoptionen (Internet Options) klicken.

2. Wählen Sie hier zunächst mit einem Mausklick im oberen Bereich des Fensters das Register Sicherheit (Security) aus.

3. Markieren Sie als nächstes den Eintrag Internet mit einem Mausklick.

4. Im unteren Teil des Fensters Internetoptionen (Internet Options) finden Sie nun entweder einen Schieberegler vor oder nur einen Text und zwei Schaltflächen.

5. Klicken Sie im Fenster Internetoptionen (Internet Options) im Register Sicherheit (Security) auf die Schaltfläche Stufe anpassen (Custom Level...). Es erscheint ein neues Fenster, in dem Sie solange scrollen, bis Sie zum Eintrag Cookies gelangst. Aktivieren Sie dort beide Einträge.

6. Zum Schluss schliessen Sie die offenen Fenster mit OK. Sie erhalten dabei noch die Abfrage, ob die Sicherheitseinstellungen wirklich geändert werden sollen. Klicken Sie hier auf Ja (Yes).
Internet Explorer 6 schließen

1. Klicken Sie im Internet Explorer zunächst auf den Menüpunkt Extras (Tools). Es erscheint ein Pulldown-Menü, in dem Sie auf Internetoptionen (Internet Options) klicken.

2. Wählen Sie hier das Register Datenschutz (Privacy) aus. Finden Sie hier unter Einstellungen keinen Schieberegler vor, klicken Sie auf die Schaltfläche Standard.

3. Schieben Sie den Schieberegler auf die Position Mittel (Medium).

4. Schliessen Sie das Fenster mit OK.
Mozilla Firefox schließen

1. Klicken Sie im Firefox Browser zunächst auf den Menüpunkt Extras (Tools). Es erscheint ein Pulldown-Menü, in dem Sie auf Einstellungen (Options) klicken.

2. Wählen Sie hier das Symbol Privacy aus.

3. Erweitern Sie den Bereich Cookies im rechten Fensterbereich.

4. Aktivieren Sie die Option Cookies akzeptieren (Enable Cookies) und stellen die Option Cookies behalten auf "solange sie gültig sind" (Accept Cookies normally).

5. Schließen Sie das Fenster mit OK.
Opera schließen

1. Klicken Sie im Opera Browser zunächst auf den Menüpunkt Extras (Tools). Es erscheint ein Pulldown-Menü, in dem Sie auf Einstellungen (Options) klicken.

2. Wählen Sie hier das Register Privatsphäre (Privacy) aus.

3. Aktivieren Sie die Option Cookies zulassen (Enable Cookies) und stellen die Optionen Normale Cookies (Normal Cookies) und Drittanbietercookies (Third party Cookies) auf "Alle Cookies annehmen" (Accept all cookies).

4. Schließen Sie das Fenster mit OK.
