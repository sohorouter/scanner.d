Die ECO-Soft GmbH erstellt seit 1994 maßgeschneiderte Software
für Unternehmen, die die Software an Ihr Unternehmen anpassen
und nicht das Unternehmen an die Software.

Die ECO-Soft GmbH ist Microsoft Small Business Specialist, als Small Business Specialist sind wir mit den Anforderungen und Bedürfnissen von kleineren und mittleren Unternehmen KMU bestens vertraut.

Die ECO-Soft GmbH erstellt individuelle Software in Salzburg und Obertrum am See.

Individuelle Software ist an die Tätigkeit des Unternehmens angepasst, sie automatisiert Arbeitsvorgänge ganz oder teilweise.
Individuelle Software vermeidet oder verringert Doppeleingaben und Eingabefehler.
Unsere Individuelle Software ist an den Zielen wirtschaftlichkeit und wirksamkeit ausgerichtet

Wir erstellen individuelle Software auf Basis folgender Technologie:
- .NET 3.5, Windows Presentation Fondation WPF, XAML, MS SQL Server
- Microsoft Office ( insbesonders Microsoft Access )
( Die Verwendung von Access als Basis für individuelle Software macht Anwendungen
auch in kleinen Unternehmen und für wenige Anwender wirtschaftlich )
- ASP.NET, ASP für Webanwendungen

Die Entwicklung individueller an das Unternehmen angepasster Software ist unsere besondere Stärke
