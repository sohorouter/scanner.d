category mail

http://gw.arge.at/webmail/

Zugriff auf das Webmail und Arge Mailsystem für den schnellen Zugriff auf persönliche Mails. Für richtige Mailverwaltung sollten Sie einen echten Mailclient installieren.

http://www.arge.at/extras/arge/webmail/

Zugriff auf das alte Webmailsystem (Mails werden nicht mit dem neuen Webmailsystem syncronisiert)!

category calendar

http://gw.arge.at/davical/

Zugriff auf den Calendar Server der Arge (jeder Mittarbeiter hat Zugriff via Netzwerk Zugangsdaten), ein Caldav Link sieht folgendermaßen aus:
http://gw.arge.at/davical/caldav.php/benutzer/home - Benutzer durch den eigenen Loginnamen ersetzen, home durch den Namen des Kalenders ersetzen. Jeder Benutzer kann mehrere Kalender erstellen. Ein Kalender kann mit einen caldavfähigen Terminkalender Programm (Thunderbird mit Lightning Addon) erstellt werden.
