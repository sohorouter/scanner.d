Bei der Konzeptionierung werden 4 Fragen geklärt:
  • Was wollen Sie auf Ihrer Homepage darstellen ?
  • Wen wollen Sie mit Ihrer Homepage ansprechen ?
  • Wie wollen Sie diese Informationen präsentieren ?
  • Wie wollen Sie diese Informationen aktuell halten (warten)?
Bei der Konzeptionierung einer Homepage wird zunächst ein Grundgerüst festgelegt.

Es wir bestimmt, welche und wieviel Informationen auf der Homepage dargestellt werden sollen. Anhand dieser Daten kann dann der erste Entwurf der HompagenavigationHomepagenavigation :

ist ein Menü von dem aus man in jeden Bereich der Hompage kommen kann.

Jeder Menüpunkt verlinkt eine Seite der Homepage.

Es kann nur aus einem Hauptmenü oder auch mit Untermenüs realisiert werden. erstellt werden.

Danach wird festgelegt, für welche Zielgruppe die Hompage ausgelegt wird,
und welche PräsentationsformenPräsentationsformen :

Die CSS - Technik..
Mit cascading style sheets werden Farben, Schriftarten und Schriftgröße sowie Abstände, Ränder und Layouts definiert.

Die Image - Technik..
Bilder geben der Seite ihren Chrakter. Sie werden für den Hintergrund von Tabellen, Blöcken, Hyperlinks verwendet.

Die Flash - Technik..
Animierte Vektorgrafiken oder Bilder geben der Hompage ein modernes Erscheinungsbild. Menüs, Formulare aber auch die gesammte Hompage kann mit Flash realisiert werden. verwendet werden.

Ein sehr wesentlicher Punkt ist die Wartung der Hompage.
Anhand der 3 Fragen wird das WartungssystemWartungssystem :

Programmiersystem..
Die Inhalte der Homepage werden mit Deklarationnen und Normgerechten TAGs von Programmieren verändert. Dabei werden die Daten vom Server huntergeladen manipuliert und wieder am Server hinaufgeladen.

Editorialessystem..
Mit einem Editor können die Inhalte der Homepage ohne Programmierkenntnisse verändern werden. Der Editor übernimmt dabei die normgerechte Syntax. Bei dieser Variante können die Daten in einer Datenbank abgelegt sein und direkt am Server manipuliert werden. definiert:
  • Wie oft müssen Daten aktualisiert werden ?
  • Wie können die Daten aktualisiert werden ?
  • Wer wird die Daten aktualisieren ?
