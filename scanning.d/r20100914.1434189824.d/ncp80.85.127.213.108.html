Lösung
fritz-box usb-speicher einhängen: sudo mount -t cifs -o username=ftpuser,password=test //192.168.178.1/MEDIONHDDrive2GO-0-1 /media/fritzbox/
web-server mit iptdfables für einen erreichbar machen: sudo iptables -F
sudo iptables -A INPUT -p tcp --dport http -s <ip> -j ACCEPT
sudo iptables -A INPUT -p tcp --dport http -j DROP
.flv zu .mp3: ffmpeg -i input.flv -vn -ar 44100 -ac 2 -ab 128k output.mp3
Prozess nach ssh-session weiterlaufen lassen: in der ssh-session:
screen
<normal arbeiten>
<strg> + <a> + <d>
exit
nächste session:
screen -r
Verzeichnis vom Webserver schützen: Passwortdatei erstellen:

htpasswd -c <pfad zu passwordfile> <user>

in /etc/apache2/httpd.conf einen Eintrag erstellen:

<Directory /var/www/DIRECTORY>
AuthType Basic
AuthName "Restricted Files"
# (Following line optional)
AuthBasicProvider file
AuthUserFile <pfad zu passwordfile>
Require user <user>
</Directory>

more info: http://httpd.apache.org/docs/2.2/howto/auth.html
