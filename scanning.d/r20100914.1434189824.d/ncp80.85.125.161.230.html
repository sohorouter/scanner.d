Front Office Internet Schnittstelle

Diese Seite ist nicht zur direkten Verwndung gedacht. Sie gibt Hinweise zur Implementierung der FO Internet-Schnittstelle auf der eingene Webseite. Alle Funktionen sind in asp.net 2.0 User Controls implementiert, alternativ dazu können die vorgefertigten Seite auf mit GET Parameter aufgerufen werden.

Beispiel 1: Verfügbarkeit aller Kategorien

Zeitraum:  
1.5.10.15.20.25.30.
EZ
EZV
DZ
DZV
Appartment

Legende:  Zimmer verfügbar   Zimmer verfügbar (Keine Anreisen)  Belegt

Integration als User-Control

<hotel:contingent runat="server" ID="contingent1" EnableViewState="false" CategoryURL="_catInfo.aspx" BookingURL="_book.aspx" />

Parameter Beschreibung
DateRange Monat der Anzeige z.B. 5/2007
Category Katgeorie z.B. EZ, fehlt dieser Parameter, werden alle Kategorien angezeigt
BookingURL URL der Seite, die bei Klick auf einen Anreisetag aufgerufen wird, es werden Kategorie, Datum und Sprache als Parameter übergeben. und. z.B. book.aspx
CategoryURL URL der Seite, die bei Klick auf einen Anreisetag aufgerufen wird. es werden Kategorie und Sprache als Parameter übergeben. z.B. CatInfo.aspx
Target Targetframe der URLs oben, z.B. _parent
Lang Sprache z.B. de, en

Es könenn die selben Parameter wie oben mit dem URL übergeben werden.


Beispiel 2: Verfügbarkeit einer einzelnen Kategorie

Zimmer Rote Wand
Suite "Rote Wand" (45 qm) mit Bad, WC, Schlafzimmer, Wohnzimmer mit Kachelofen, Balkon zum Ort
Zeitraum:  
1.5.10.15.20.25.30.

Integration als User-Control

<hotel:contingent runat="server" ID="contingent2" EnableViewState="false" BookingURL="_book.aspx" Category="ROW" />

Es werden die selben Paramete wie oben verwendet, es muss der Parameter Cat angegeben werden.

Integration als Popup

» Hier zur Online-Buchung "Rote Wand"

<a href="_catInfo.aspx?category=ROW" target="_blank" >Hier zur Online-Buchung</a>

Es werden die selben Paramete wie oben verwendet, es muss der Parameter Cat angegeben werden.


Beispiel 3: Packages und direktes Buchen

Aktionswoche HP im grossen Zimmer vom 16.3. - 20.3.2007 890,00 »Buchen!

Integration als Popup

<a href='_book.aspx?date=16.3.2007&date1=20.3.2007&Cat=ROW&Arr=HP 7&Package=1'> »Buchen!</a>

Parameter Beschreibung
Date Anreisetag, das Datum wird immer als tt.mm.yyy übergeben, z.B. 25.1.2007
Date1 Abreisetag, das Datum wird immer als tt.mm.yyy übergeben, z.B. 27.1.2007
Cat Vorbelegung Kategorie, z.B. DZ
Arr Vorbelegung Arrangement, z.B. HP
Package Ist dieser Parameter gleich 1, werden die Auswahlfelder für Datum, Kategorie und Arrangement gesperrt, es ist dann keine Änderung der Vorbelegung möglich.
