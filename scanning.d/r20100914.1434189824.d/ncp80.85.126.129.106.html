Telsec Management Software home

This part of the software, the Viewer section, is a complete webbased program which generates statistics for all the data that is collected from TCU's on one or more locations.

Quick Help
The mouse cursor can be used as a help tool by pointing the mouse to objects* on the page. When help is available for the object* on which the mouse cursor is located a "Quick Help" box will appear with information about using the object*.
*An object can be anything (links, text, images, numbers.. etc)on the page.

Location overview
Click on the "Location" button in the menu to open the page with all the locations. To open the detailed statistics of a location click on the name of the location.
