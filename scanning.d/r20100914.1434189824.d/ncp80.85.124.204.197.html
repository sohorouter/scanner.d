Looop Test Channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover pear.devserver.looop.at
Listing available packages: 
pear remote-list -c looop
Installing a package: 
pear install looop/package_name
Installing a specific version/stability: 
pear install looop/package_name-1.0.0
pear install looop/package_name-beta
Receiving updates via a feed: 
http://pear.devserver.looop.at/feed.xml
